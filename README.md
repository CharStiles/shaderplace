# Shader Place
A realtime livecode GLSL editor

## Requirements
- node.js

## Quickstart!
```
$ git clone https://github.com/CharStiles/shaderplace.git
$ cd shdaerplace
$ npm i
$ npm start
```
go to http://localhost:8080/
